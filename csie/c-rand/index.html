# rand()

產生介於 0~RAND_MAX (=32767) 之間的隨機亂數，需要透過 srand () 來初始化

# srand()

透過傳進來的 seed 初始化 rand ()，seed 若不改變則每次所產生的亂數皆會相同。

# time.h

# time()

time(NULL) 會回傳自 1970 年午夜到現在所經過的秒數，可以用作 srand 的 seed，每次執行的 seed 皆會不一樣
